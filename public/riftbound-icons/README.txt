Riftboundle — Discord-sourced icons
====================================

Filenames match Riftcodex rule text tokens without colons, e.g. :rb_energy_3: → rb_energy_3.webp.

Shipped tokens (see manifest.json)
----------------------------------
rb_might.webp, rb_exhaust.webp
rb_rune_body … rb_rune_order, rb_rune_rainbow
rb_rune_unknown.webp — UI-only: rules + cost plate while domain hint is off (not a Riftcodex token).
rb_energy_0.webp … rb_energy_9.webp

Not shipped (optional)
----------------------
rb_recycle — referenced in some UIs; add rb_recycle.webp when you have art; until then :rb_recycle: falls back to a monospace placeholder in rules text.

Extras (not :rb_*: in card text scan)
-------------------------------------
icon_gear_white.webp, icon_spell_white.webp, icon_unit_white.webp — optional UI/type icons (not wired to :rb_*: tokens).

Original Discord CDN used ?size=128; you can re-download at ?size=96 or ?size=40 for smaller files.
